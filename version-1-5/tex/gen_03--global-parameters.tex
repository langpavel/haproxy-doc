\chapter{Global parameters}

Parameters in the \emph{global} section are process-wide and often OS-specific. They
are generally set once for all and do not need being changed once correct. Some
of them have command-line equivalents.


The following keywords are supported in the \emph{global} section :

\begin{verbatim}
 * Process management and security
   - chroot
   - daemon
   - gid
   - group
   - log
   - log-send-hostname
   - nbproc
   - pidfile
   - uid
   - ulimit-n
   - user
   - stats
   - node
   - description
   - unix-bind
\end{verbatim}

\begin{verbatim}
 * Performance tuning
   - maxconn
   - maxconnrate
   - maxpipes
   - noepoll
   - nokqueue
   - nopoll
   - nosepoll
   - nosplice
   - spread-checks
   - tune.bufsize
   - tune.chksize
   - tune.http.maxhdr
   - tune.maxaccept
   - tune.maxpollevents
   - tune.maxrewrite
   - tune.pipesize
   - tune.rcvbuf.client
   - tune.rcvbuf.server
   - tune.sndbuf.client
   - tune.sndbuf.server
\end{verbatim}

\begin{verbatim}
 * Debugging
   - debug
   - quiet
\end{verbatim}
